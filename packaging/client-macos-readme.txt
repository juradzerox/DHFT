DHFT Client for macOS

Open DHFT Client.app.

The app opens a local browser control panel. Enter the server address, choose a
destination folder, enter the shared token, and press Start.

Server address format:
SERVER_IP_OR_DNS:9811

If macOS blocks the unsigned app, right-click it, choose Open, then confirm.

The client supports pause, stop, resume, MD5 verification, parallel workers, and
live upload/download plus disk read/write speeds.
