DHFT Server for Windows

Double-click DHFT-Server-Windows.exe.

The app opens a local browser control panel. Choose the folder to upload, set
the listen address, enter a shared token, and press Start.

Recommended listen address:
:9811

Allow TCP port 9811 through Windows Firewall. The client must connect directly
to this Windows machine's public IP, private VPN IP, or DNS name.

The token is authentication, not encryption. Use a VPN or encrypted tunnel for
sensitive transfers over the public internet.
